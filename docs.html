Authentication

All API requests require an API key sent in the header. Get your key from Dashboard → Settings → API Keys.

curl -X GET "https://api.jamilformula.com/v3/formulas/search" \ -H "Authorization: Bearer fa_live_sk_..." \ -H "Content-Type: application/json"

⚠️ Security: Never expose your key in frontend code. Always use environment variables.

Formula search

AI-powered smart search for any chemical formula using natural language.

POST /formulas/search

Parameters

Field Type Description
querystringProduct name to search for
categorystringCategory (cosmetics, detergents, ...)
countrystringCountry code for compliance check (ISO-2)
languagestringResponse language (ar, en, fr, ...)

Example request

// JavaScript / Node.js const response = await fetch('https://api.jamilformula.com/v3/formulas/search', { method: 'POST', headers: { 'Authorization': 'Bearer ' + process.env.FORMULA_AI_KEY, 'Content-Type': 'application/json' }, body: JSON.stringify({ query: 'natural shampoo for dry hair', category: 'cosmetics', country: 'SA', language: 'en' }) }); const data = await response.json(); console.log(data.formulas);

Example response

{ "id": "FA-2026-04891", "name": "Natural shampoo for dry hair", "category": "cosmetics", "confidence": 0.96, "components": [ { "name": "Sodium Laureth Sulfate", "cas": "68585-34-2", "percentage": 14.0, "function": "Primary Surfactant" }, // ... 8 more components ], "compliance": { "SA": true, "EU": true } }

Regulatory compliance check

Verify that a formula complies with regulations in any country in the world.

POST /compliance/check
# Python import requests response = requests.post( 'https://api.jamilformula.com/v3/compliance/check', headers={'Authorization': f'Bearer {os.environ["FORMULA_AI_KEY"]}'}, json={ 'formula_id': 'FA-2026-04891', 'countries': ['SA', 'AE', 'EG', 'EU', 'US'] } ) result = response.json() print(f"Compliant in {result['compliant_countries']} of {result['total']} countries")

Extract from a book

Upload a book and extract formulas from it programmatically.

POST /books/extract
# Upload book and extract formulas curl -X POST "https://api.jamilformula.com/v3/books/extract" \ -H "Authorization: Bearer fa_live_sk_..." \ -F "file=@./CosmeticFormulary.pdf" \ -F "language=auto" \ -F "complete_missing=true"
